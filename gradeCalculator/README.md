# GradeCalculator

## How to build
1. Go to project root directory
2. mvn clean package

## How to run test
1. Go to project root directory
2. mvn clean test

## How to run
1. Go to project root directory
2. mvn clean package
3. java -cp target/gradeCalculator-1.0-SNAPSHOT.jar edu.csus.csc131.gradeCalculator.App
